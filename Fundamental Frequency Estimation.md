# Fundamental Frequency Estimation



### 评估基频F0的方法
1. 自相关法： R(T) = n=0~L-1的求和(S[n]S[n-T])，T与fs/f0相关联，取能使R(T)最大的f0就是评估所得的基频，我们认为f0在L内是不变的。
2. 差分放分方程法
3. 闭环评估：每一个可能的结果都会被评估一遍，从中选出最优，精度高，复杂度高
4. 基频范围在50到400Hz之间

### 用LTP对基频建模
1/P(z) =1 / ( 1 - y.z^-T )

y表示增益，标示延时，及就是每隔一段时间，就有一个冲击，所以时延和增益就是冲击串模型的两个参数
